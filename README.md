# :helicopter: Hopter（竹蜻蜓）

Hopter is a lightweight Java MVC framework.
## Features

- [x] IoC
- [x] MVC
- [x] AOP
- [x] Transaction
- [x] File upload
- [x] Security
- [x] SOAP
- [x] REST

## Acknowledgments

> 参考自书籍 [《架构探险》](https://book.douban.com/subject/26593466/)，Hopter 的第一版基本就是书中 Smart Framework 的翻版。后续会不断改进，争取在原框架上加入新的特性。
